Implementé el Flyweight Pattern para optimizar la memoria y el rendimiento en el sistema de partículas. En lugar de crear una nueva instancia para cada partícula, las características comunes como el material, color y velocidad se agrupan en una sola clase llamada ParticleType. Usé un método estático GetSharedParticle() para asegurarme de que las partículas con los mismos atributos compartieran una sola instancia de ParticleType.
